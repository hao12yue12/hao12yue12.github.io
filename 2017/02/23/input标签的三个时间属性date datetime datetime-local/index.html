input标签的三个时间属性date datetime datetime-local

2017-02-23

  • 首先说明的是这三个标签都是HTML5提出的新属性,在firefox和IE中都不支持,接下来的使用中不考虑兼容
  • 三个标签的区别为:
    • date表示日期,在支持该标签的chrome浏览器中格式为—-/–/–,并且有选择日期的复选框,选择日期操作比较方便
    • datetime可以表示日期和时间和时区,但是已经被废弃,目前主流浏览器都不支持
    • datetime-local可以表示日期时间

  • 默认值为现在的时间,均使用js实现

  • 后端解析时间
