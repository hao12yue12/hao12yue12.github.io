shadowsocks-libev客户端安装与配置

2017-03-02

目前翻墙的方案有:fqrouter goagent shadowsocks 自由门 VPN,其中大家应用最广的应该就是shadowsocks了,他是一个开源的Socks5代理.而shadowsocks-libev是shadowsocks的一个版本,该版本用c语言开发.特点是内存占用小（600k左右），低 CPU 消耗，甚至可以安装在基于 OpenWRT 的路由器上。

  • Ubuntu下安装shadowsocks-libev

  • 配置
    客户端配置文件可以在任意目录下建立,推荐在/etc/shadowsocks-libev/config.json
    格式为:

说明如下

server：主机域名或者IP地址，尽量填IP
server_port：服务器监听端口
password：密码
method：加密方式 默认为table,其他有rc4,rc4-md5,aes-128-cfb, aes-192-cfb, aes-256-cfb,bf-cfb, camellia-128-cfb, camellia-192-cfb,camellia-256-cfb, cast5-cfb, des-cfb

  • 启动

查看是否正确启动并监听相应端口,看到ss-server进程LISTEN正确的端口就表示成功:

netstat -lnp | grep ss-server
