配置级别

一般git的配置等级可以分为系统、全局(当前用户)、当前仓库三个级别
分别配置命令为:
系统级别:配置文件在/etc/gitconfig 需加参数--system
全局级别:配置文件在~/.gitconfig或~/.config/git/config配置需加参数--global
仓库级别:配置文件在.git/config无需加参数

覆盖级别仓库>全局>系统
