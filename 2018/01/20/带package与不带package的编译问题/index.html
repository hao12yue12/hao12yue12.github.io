带package

  • 包要对应上级文件夹名(包文件夹)，所以对类进行解释时要以 packagename.classname 的形式
  • 所以要先切换到包文件的上级目录在用 java 命令，没有包名对应的文件夹则可以创建包文件夹把.class文件放进去
  • 或者用编译时以 javac -d . xxx.java 的形式自动生成包文件夹(包含.class文件，此时.java与.class不在同一文件夹)
