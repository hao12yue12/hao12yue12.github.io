Maven使用入门

编写 POM

Maven 项目的核心就是 pom.xml。
POM（Project Object Model项目对象模型）包含定义了项目如何构建、声明项目依赖等基本信息。
以下是 HelloWorld 项目中的 pom.xml，路径为 hello-world/pom.xml。

可以看出实际的项目代码与 POM 基本无关，项目升级至于修改 POM，而日常开发则只需修改 Java 代码。

运行命令 mvn clean test。实际执行步骤：

  1. clean:clean 清理 target 目录。
  2. resources:resources 寻找主项目资源。
  3. compiler:compile 编译主项目。
  4. resources:testResources 寻找测试项目资源。
  5. compiler:testCompile 编译测试代码。
  6. surefire:test 任务运行测试。
